試作・参考デザイン

訪問看護向け 補助金・支援制度リファレンス

事業所の所在地（都道府県・市区町村）を選ぶと、公式ページへ進むための候補を絞り込みます。 実際の適用可否は各公募要領・担当窓口での確認が必要です。

地域を選ぶ

市区町村は全国約1,900件です。都道府県選択後、検索欄で名前を絞り込んでから選ぶと探しやすいです。候補一覧表示後は、地域を変えると自動で再計算されます（初回のみ「候補を表示」が必要です）。

ご利用上の注意

  • 本ページは参考・プロトタイプです。掲載内容は最終確認日時点のメモであり、法的・税務上の助言ではありません。
  • 制度の正式名称・要件・締切は必ず公式サイト・公募要領で確認してください。
  • 訪問看護に特化した制度だけでなく、「中小企業」「IT導入」など事業形態次第で検討余地のある制度も候補に含める設計です。
  • 各カードの「複数サイトから拾った参考情報」は、登録したURLに対してローカルで取得したタイトル・メタ説明の抜粋です（全文ではありません）。取得できないURLもあります。
  • 都道府県・市区町村ごとの独自助成は年度・名称が変わりやすく、データ上は全都道府県・全市区町村を網羅していません。未掲載地域は、国の自治体リンク集から公式サイトへ辿り、サイト内検索で「訪問看護」「看護ステーション」「事業者」「助成」等を試してください。
  • 制度の最終確認・申請判断はすべて利用者個人の責任です。本ページは探索の手がかりを広げるための参考であり、助言や保証ではありません。

候補一覧

    条件に合う候補がデータ上はありません。公式の県・市の事業者向けページを直接ご確認ください。
